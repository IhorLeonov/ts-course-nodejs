nodemon.json:

- "watch": ["src"]. - Цей параметр вказує nodemon стежити за змінами у всіх файлах всередині папки "src". 
Якщо будь-який файл у цій папці буде змінено, nodemon автоматично перезапустить сервер.

- "ext": "ts". - Цей параметр говорить nodemon стежити тільки за файлами з розширенням ".ts" (TypeScript-файлами).

- "exec": "ts-node -r tsconfig-paths/register src/main.ts". - Цей параметр вказує команду, яку nodemon має виконати під час запуску сервера. 
У цьому випадку використовується ts-node, інструмент, який дає змогу запускати TypeScript-файли напряму (без попередньої транспіляції у JavaScript). 
Прапор "-r tsconfig-paths/register" означає, що ми хочемо зареєструвати модуль "tsconfig-paths" перед запуском файлу "src/main.ts". 
Цей модуль дає змогу резолвити шляхи, зазначені у файлі конфігурації TypeScript (tsconfig.json), у рантаймі.
